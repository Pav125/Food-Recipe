# SundayFlavors

Welcome to SundayFlavors, a website for exploring and discovering delicious recipes! Built using React.js and styled with Tailwind CSS, SundayFlavors integrates an external API from Edamam to fetch ingredients and images of searched items, providing users with a delightful culinary experience.

## Features

- **Recipe Search**: Explore a vast collection of recipes by searching for ingredients or dish names.
- **Ingredient Details**: View detailed information about each ingredient, including nutritional values and serving sizes.
- **Image Gallery**: Visualize recipes with high-quality images fetched from the Edamam API.
- **Responsive Design**: Enjoy a seamless browsing experience across various devices, thanks to the website's responsive design.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **API Integration**: Edamam API
- **Design**: Figma

## Setup Instructions

To run this project locally, follow these steps:

1. **Clone the Repository**

    ```sh
    git clone https://github.com/Pav125/Food-Recipe.git
    cd Food-Recipe
    ```

2. **Install Dependencies**

    ```sh
    npm install
    ```

3. **Start the Development Server**

    ```sh
    npm start
    ```

4. **Access the Website**

    Open your browser and navigate to `http://localhost:3000`.

5. **Create a .env file**

    Add the App ID and APP KEY from the Edamam
